# My Multipage Website


## Project Purpose
This is a multipage website showcasing a portfolio/product/blog with multiple pages including Home, About, Services, Gallery, and Contact. The goal is to demonstrate responsive design, interactivity, and proper website structure.


## Structure
- `index.html` - Home page
- `about.html` - About page
- `services.html` - Services offered
- `gallery.html` - Image gallery
- `contact.html` - Contact form
- `css/style.css` - Styles for the website
- `js/script.js` - JavaScript interactivity
- `images/` - All website images
